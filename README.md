# Project KBTA: MountainGo Bromo Trip Website

Semua file projek website pemesanan trip Bromo berada di dalam direktori **`mountaingo/`**.

Silakan merujuk ke panduan instalasi lengkap di sini:
👉 **[mountaingo/README.md](file:///c:/KUMPULAN%20SMT%204/KUMPULAN%20TUGAS%20MATKUL/projectkbta/mountaingo/README.md)**

### Cara Cepat Menjalankan Proyek:
1. Buka terminal di folder `mountaingo`.
2. Jalankan perintah:
   ```bash
   pip install -r requirements.txt
   python app.py
   ```
3. Buka browser di **[http://127.0.0.1:5000](http://127.0.0.1:5000)**.
